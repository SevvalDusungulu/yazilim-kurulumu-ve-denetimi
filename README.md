Yazılım Kurulumu ve Denetimi – Bilgilendirici İçerik
1. Amaç

Bu repository, bir bilgisayar sisteminde yazılım kurulumu, yazılım denetimi ve temel sistem kontrollerinin nasıl yapıldığını göstermek amacıyla hazırlanmıştır. İçerik, özellikle yeni başlayanlar için rehber niteliğindedir.

-Kullanılan İşletim Sistemi

Windows 11

Linux (Ubuntu – VirtualBox sanal makine)

-Yazılım Kurulumu Örneği
Örnek Yazılım: Python

Python, günümüzde veri bilimi, yapay zeka ve yazılım geliştirme alanlarında yaygın olarak kullanılan bir programlama dilidir.

Kurulum adımları (Windows):

Resmi web sitesinden indirme yapılır

Kurulum sırasında “Add Python to PATH” seçeneği işaretlenir

Kurulum tamamlanır

Kurulumdan sonra aşağıdaki komut ile kontrol yapılır:
python --version
Bu komut, Python’un başarıyla kurulup kurulmadığını gösterir.
Windows CMD ile yapılan kontroller
Komut     Açıklama
dir   	   Bulunulan dizindeki dosyaları listeler
cd	       Dizin değiştirir
mkdir	     Yeni klasör oluşturur
echo	     Dosya içine metin yazar
type	     Dosya içeriğini gösterir
copy	     Dosya kopyalar
del    	   Dosya siler
ipconfig	 Ağ yapılandırmasını gösterir
systeminfo Sistem bilgilerini listeler
cls	Komut  ekranını temizler

Bu komutlar sayesinde sistem durumu ve yazılım işleyişi denetlenebilir.

Linux Terminal ile yapılan kontroller
Komut	   Açıklama
pwd    	Bulunulan dizini gösterir
ls	    Dosya ve klasörleri listeler
mkdir	  Yeni klasör oluşturur
cd	    Dizin değiştirir
touch	  Dosya oluşturur
echo	  Dosya içine veri yazar
cat	    Dosya içeriğini görüntüler
cp	    Dosya kopyalar
rm	    Dosya siler
